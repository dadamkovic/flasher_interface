\begin{DoxyParagraph}{}
Demonstrates a gated recurrent unit (GRU) example with the use of fully-\/connected, Tanh/\+Sigmoid activation functions.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Model definition\+:}

\end{DoxyParagraph}
\begin{DoxyParagraph}{}
GRU is a type of recurrent neural network (RNN). It contains two sigmoid gates and one hidden state. 
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
The computation can be summarized as\+: 
\begin{DoxyPre}z[t] = sigmoid( W\_z {$\cdot$} \{h[t-\/1],x[t]\} )
r[t] = sigmoid( W\_r {$\cdot$} \{h[t-\/1],x[t]\} ) 
n[t] = tanh( W\_n {$\cdot$} [r[t] {$\times$} \{h[t-\/1], x[t]\} ) 
h[t] = (1 -\/ z[t]) {$\times$} h[t-\/1] + z[t] {$\times$} n[t] \end{DoxyPre}
 
\end{DoxyParagraph}

\end{DoxyParagraph}

\end{DoxyParagraph}
\begin{DoxyParagraph}{}
The buffer is allocated as\+: 
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
$\vert$ reset $\vert$ input $\vert$ history $\vert$ update $\vert$ hidden\+\_\+state $\vert$ 
\end{DoxyParagraph}
\begin{DoxyParagraph}{}

\end{DoxyParagraph}
