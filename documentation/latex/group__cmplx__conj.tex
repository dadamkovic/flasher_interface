\doxysubsection{Detailed Description}
Conjugates the elements of a complex data vector.

The {\ttfamily p\+Src} points to the source data and {\ttfamily p\+Dst} points to the where the result should be written. {\ttfamily num\+Samples} specifies the number of complex samples and the data in each array is stored in an interleaved fashion (real, imag, real, imag, ...). Each array has a total of {\ttfamily 2$\ast$num\+Samples} values. The underlying algorithm is used\+:


\begin{DoxyPre}
for(n=0; n<numSamples; n++) \{
    pDst[(2*n)+0)] = pSrc[(2*n)+0];     // real part
    pDst[(2*n)+1)] = -\/pSrc[(2*n)+1];    // imag part
\}
\end{DoxyPre}


There are separate functions for floating-\/point, Q15, and Q31 data types.
