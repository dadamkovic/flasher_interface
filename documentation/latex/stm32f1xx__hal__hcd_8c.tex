\doxysubsection{Detailed Description}
HCD HAL module driver. This file provides firmware functions to manage the following functionalities of the USB Peripheral Controller\+: 

\begin{DoxyAuthor}{Author}
MCD Application Team
\begin{DoxyItemize}
\item Initialization and de-\/initialization functions
\item IO operation functions
\item Peripheral Control functions
\item Peripheral State functions
\end{DoxyItemize}
\end{DoxyAuthor}
\begin{DoxyVerb}==============================================================================
                  ##### How to use this driver #####
==============================================================================
[..]
  (#)Declare a HCD_HandleTypeDef handle structure, for example:
     HCD_HandleTypeDef  hhcd;

  (#)Fill parameters of Init structure in HCD handle

  (#)Call HAL_HCD_Init() API to initialize the HCD peripheral (Core, Host core, ...)

  (#)Initialize the HCD low level resources through the HAL_HCD_MspInit() API:
      (##) Enable the HCD/USB Low Level interface clock using the following macros
           (+++) __HAL_RCC_USB_OTG_FS_CLK_ENABLE();
      (##) Initialize the related GPIO clocks
      (##) Configure HCD pin-out
      (##) Configure HCD NVIC interrupt

  (#)Associate the Upper USB Host stack to the HAL HCD Driver:
      (##) hhcd.pData = phost;

  (#)Enable HCD transmission and reception:
      (##) HAL_HCD_Start();\end{DoxyVerb}
