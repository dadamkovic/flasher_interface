\doxysubsection{Detailed Description}
Multiplies a complex vector by a real vector and generates a complex result. The data in the complex arrays is stored in an interleaved fashion (real, imag, real, imag, ...). The parameter {\ttfamily num\+Samples} represents the number of complex samples processed. The complex arrays have a total of {\ttfamily 2$\ast$num\+Samples} real values while the real array has a total of {\ttfamily num\+Samples} real values.

The underlying algorithm is used\+:


\begin{DoxyPre}
for(n=0; n<numSamples; n++) \{
    pCmplxDst[(2*n)+0] = pSrcCmplx[(2*n)+0] * pSrcReal[n];
    pCmplxDst[(2*n)+1] = pSrcCmplx[(2*n)+1] * pSrcReal[n];
\}
\end{DoxyPre}


There are separate functions for floating-\/point, Q15, and Q31 data types.
